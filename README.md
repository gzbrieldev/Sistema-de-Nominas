# Sistema de Nómina en C# (aplicación de consola)
C# .NET 8.0
## Descripción
Este proyecto implementa un sistema de nómina en C#, con una arquitectura orientada a objetos y separación de responsabilidades.  
Permite registrar, buscar, actualizar y listar empleados de distintos tipos:

- Empleado Asalariado
- Empleado por Horas
- Empleado por Comisión
- Empleado Asalariado + Comisión

Cada empleado tiene un **Número de Seguro Social (NSS)** único e inmutable.

---

## Funcionalidades
- Registrar empleados nuevos (incrementando un contador para llevar la cuenta).
- Buscar empleados por NSS.
- Actualizar datos de empleados existentes.
- Listar todos los empleados registrados.
- Rendimiento.
